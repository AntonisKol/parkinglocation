# Where is the car parked? Frontend

This is a React and Vite frontend for saving and viewing a car parking location.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Configuration

The app uses `https://backend-adwr.onrender.com` by default. To use another backend, create `frontend/.env`:

```bash
VITE_BACKEND_URL=http://localhost:8000
```
